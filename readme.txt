== Description ==

Add fullscreen slideshow backgrounds to any post or page using MaxImage 2.0.

== Installation ==

1. Upload maxi-bg folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on "Add Slideshow Background" on the "Maxi BG" admin tab
4. Copy the shortcode and paste to a post or page


== Frequently Asked Questions ==

= How to display it in post or page? =
Use the shortcode '[maxibg id="ID"]'
